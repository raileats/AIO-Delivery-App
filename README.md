# Vendor Orders Aggregator

This project combines vendor orders (Zomato, Swiggy, etc.) into a single dashboard.

## Structure
- `backend/` → Express + IMAP + SQLite
- `frontend/` → React dashboard

## Deployment
- Backend → Railway / Render
- Frontend → Vercel

## Setup
1. Run backend:
   ```bash
   cd backend
   npm install
   npm start
   ```
2. Run frontend:
   ```bash
   cd frontend
   npm install
   npm start
   ```
